# 智能课程表 Pro · UI 设计规范（v1.0）

> 配套高保真原型：`prototype.html`（自包含、可交互，左侧切换屏幕，带「状态」标记的页面可切换 默认/空/加载/错误 四种状态）。
> 设计目标：现代、低认知负担、年轻活力；主色冷杉青绿 + 暖琥珀强调，纸感暖灰背景，12 色无障碍课程调色板。

### 1.2 字体排版
- **正文字体**：`"Noto Sans SC", system-ui, "PingFang SC", "Microsoft YaHei"` — 中文跨平台一致。
- **数字 / 英文显示**：`"Space Grotesk"` — 节次、时间、周次数字更具设计感。
- **字号比例（8pt 友好的 1.25 模数）**：

| 级别 | 字号 | 字重 | 用途 |
| --- | --- | --- | --- |
| 大标题 | 28px | 900 | 品牌页 / 空状态主标 |
| 页标题 | 20px | 700 | 周/日课表标题 |
| 区块标题 | 18px | 700 | 规划 Hero、弹窗标题 |
| 正文 | 15px | 500 | 列表、说明 |
| 辅助 | 13px | 400 | 次要信息 |
| 标注 | 12px | 400 | 提示、脚注 |
| 数字 | 14px | 600 | Space Grotesk，节次/时间 |

- 行高 1.5（正文），标题 1.2–1.3。

### 1.3 间距系统（4pt 基准）
所有内边距 / 外边距 / 组件间隙均取自该尺度，确保视觉节奏统一：

| Token | 值 | 典型用途 |
| --- | --- | --- |
| `--s1` | 4px | 图标与文字微距 |
| `--s2` | 8px | 小组件内距、chip 内边距 |
| `--s3` | 12px | 卡片内边距、表单标签间距 |
| `--s4` | 16px | 区块外边距、列表行内边距 |
| `--s5` | 20px | 弹窗内边距 |
| `--s6` | 24px | 页面区块间距 |
| `--s8` | 32px | 顶部栏下间距 |
| `--s12` | 48px | 空/错误页纵向留白 |
| `--s16` | 64px | 大区块分隔 |

### 1.4 圆角 & 阴影 & 动效
- **圆角**：`--r-sm`10 / `--r-md`14 / `--r-lg`20 / `--r-xl`28 / 胶囊 `999px`。手机外壳 46px。
- **阴影**（柔和、低饱和，避免通用 AI 发光感）：
  - `--sh-1`：卡片静止（1px 轻投影）
  - `--sh-2`：悬浮 / 弹层
  - `--sh-3`：手机框 / 全局错误页
  - `--sh-brand`：主色 CTA（青绿光晕，非霓虹）
- **动效**：状态变化统一用 `cubic-bezier(.22,1,.36,1)`（ease-out-quart）；时长 160ms（快）/ 300ms（常规）；底部抽屉 `slideUp` 350ms，Toast `toastIn` 400ms；**尊重 `prefers-reduced-motion`**（骨架屏、加载圈仍保留但去位移）。

## 2. 组件库与间距规范

| 组件 | 关键规范 |
| --- | --- |
| **按钮** | 主 `--brand-600` 白字 + 光晕；强调 `--amber-500` 深字；次 `--ghost` 描边；危险 `--err`。高 44–56px，圆角 `--r-sm`，内边距 `13px×20px`（块级全宽）。`:active` 缩放 .97。 |
| **输入框** | 高 46px，圆角 `--r-sm`，边框 `1.5px --ink-200`；聚焦 → `--brand-500` 描边 + `0 0 0 4px --brand-50` 光晕；错误态红边 + 红晕 + 下方红字提示。 |
| **芯片 Chip** | 圆角胶囊，内边距 `5px×11px`；品牌/琥珀/成功/错误四态语义色，用于单双周、AI 入口、空闲标注。 |
| **单双周角标** | `单`=靛蓝底、`双`=橙底小标签，置于课程块右上角；错周自动隐藏（F-06）。 |
| **课程色块** | 浅底 + 左侧 3px 深条纹；内含课程名（11.5px/700）、地点、教师（带图标）；跨节次时整块合并。 |
| **底部抽屉 Sheet** | 从底部滑入，圆角 `28/28/0/0`，遮罩 `rgba(28,25,23,.42)` 点击关闭；顶部手柄 38×4px。 |
| **Toast** | 底部居中，深底白字，含操作（如「撤销」）；出现 400ms 上滑。 |
| **骨架屏** | 浅灰流光（`sh` 动画 1.3s），形状与最终内容一致。 |
| **空状态** | 居中插画（SVG）+ 主标 + 说明 + 主/次 CTA，引导下一步。 |
| **错误页** | 居中插画 + 可重试按钮，绝不白屏（FR-S3）。 |

## 3. 页面级交互细节与状态覆盖

> 标注格式：`[屏幕] 交互 → 细节（间距/反馈）`。所有「带状态」页面在原型中可一键切换 默认/空/加载/错误。

### 3.1 周课表（F-01）— 支持 4 态
- **布局**：顶部栏（标题 + 搜索/AI 图标，间距 `--s4`）→ 周序条（7 天，当前日 `--brand-600` 实心高亮，gap 5px）→ 7×N 网格（时间列 34px + 7 列，行高 50px，gap 5px，min-width 560px 可横滑）。
- **课程块**：按 `--cN` 着色；单双周角标；地点/教师带图标；跨节次合并。
- **当前日列**：浅青底 `--brand-50` + 浅边框。
- **无课单元格**：显示「空闲」浅灰占位（`--ink-300`）。
- **空态**：插画 + 「你的课表还是空的」+ AI 导入 / 手动添加双 CTA。
- **加载态**：时间列 + 课程格全部骨架屏。
- **错误态**：友好错误页 + 「重新加载」。

### 3.2 日课表（F-02）— 支持 4 态
- **布局**：时间轴（左侧 54px 时间轨 + 节点圆点 + 连接线）→ 卡片列表（课程卡左侧 4px 深条纹对应课程色）。
- **空闲块**：灰色卡 + 太阳图标 + 「可自由安排自习 / 休息」+ 时长（如「空闲 2小时35分」）。
- **顶部横幅**：双周时提示「单周课程已自动隐藏」（F-06）。
- **空态**：「今天全天空闲 🎉」+ 生成今日规划 CTA。**加载 / 错误**同周视图规范。

### 3.3 手动添加 / 编辑课程（F-03 / F-04）
- 底部抽屉表单：课程名（必填 `*`，红星）／星期（7 段选择器）／节次或起止时间（二选一）／地点（可空，空显「—」）／教师（可空）／单双周（每周·单周·双周 三选）／课程颜色（6×2 调色板，选中态深色描边 + 内白圈）。
- **实时校验**：空必填 / 非法字符 → 即时红边 + 红字，不写入脏数据（FR-S1）。
- **冲突预警**（F-03 业务规则）：与同周次课程时间重叠 → **预警而非禁止**（便于调停课），横幅提示「与『数据结构』时间重叠，将同时显示」。
- 编辑复用同一表单，保留课程 ID（F-04）。

### 3.4 删除确认 + 撤销（F-05）
- 课程详情卡（色块 + 名称 + 星期节次 + 地点/教师）→ 删除按钮（危险红）。
- **二次确认**抽屉：警示图标 + 「删除『数据结构』？可在 30 秒内撤销」+ 受影响条数说明。
- **软删除 + 30s 撤销**：确认后 Toast「已删除…」含「撤销」按钮；批量按名称匹配并提示条数。

### 3.5 AI 一键导入（F-08）— 多步流程，支持降级
- **入口**：粘贴文本 / 上传截图 两张大卡片（图标 + 说明）。
- **粘贴**：文本框（≤8000 字）+ 解析按钮 + 「未配置 Key 将用本地免费模型」横幅。
- **截图**：虚线拖拽区 + 缩略图预览。
- **解析中**：居中旋转环 + 进度文案（文本 <8s / 截图 <15s，超时降级）。
- **预览确认**：列表展示解析结果（✔ 通过 / ⚠ 可疑字段标红如「教师缺失」），顶部「置信度 92%」+ 标红横幅；提供 **重新解析 / 手动修正 / 确认导入** 三选项，确认前不覆盖现有课表。
- **失败降级**（踩坑① / FR-S2）：超时 / 500 / 格式异常 → 友好错误页 + 「改用文本手动录入 / 重新解析」，**绝不白屏**。

### 3.6 空闲规划 + AI 规划助手（F-09）— 支持 4 态 + 本地模板兜底
- **Hero**：模式切换「自习 / 考研 / 考公」（胶囊切换，选中白底 + 主色字）。
- **空闲时段列表**：星期 + 时间段 + 时长（Space Grotesk 加粗主色）。
- **AI 复习计划**：任务卡（科目 + 时段 + 时长 chip + 建议）。
- **降级兜底**（业务规则）：AI 不可用时自动套用预设模板（考研＝政治/英语/专业课轮动），横幅告知「已切换为本地模板」；结果可「加入日历提醒 / 导出 .ics」。
- **空态**：先建课表再规划。**加载 / 错误**同规范（错误态直接给本地模板入口）。

### 3.7 .ics 日历导出（F-10）
- 步骤① 选择范围（整学期 18 周 / 本周 / 所选周，段选择器）；
- 步骤② 包含内容（课前 15 分钟提醒、地点/教师备注，开关）；
- 文件卡（文件名 `timetable_2025spring.ics` + 事件数 + 大小）；
- 下载按钮 → 引导「iPhone 文件 App 点击导入 / Android 用 Google 日历打开」；
- 单双周**拆分为独立 VEVENT**（兼容性最佳，避免 RRULE 串周）。

### 3.8 设置（学期 / AI Key）& 全局错误兜底（FR-S3）
- **设置**：学期起始日 / 总周数 / 周起始日；AI Key（本地加密、仅本机、不落日志）；JSON 备份恢复；.ics 导出入口。
- **全局 Error Boundary**：任何未预期错误 → 「出错了，但你可以重试」页（非白屏）+ 重新加载 / 导出备份。

## 4. 响应式与跨端（PWA）
- **移动优先**（核心场景：iOS Safari 15+ / Android Chrome 100+）。手机外壳 390×844 为设计基准。
- **断点**：640 / 768 / 1024 / 1280（同前文框架）；周课表网格在窄屏 `min-width:560px` 横向滚动，不被压缩隐藏。
- **容器查询**：卡片/列表在平板以上可并排（如设置分组两列）。
- **安全区**：底部 Tab Bar 与 FAB 预留 `env(safe-area-inset-bottom)`。
- **可添加主屏幕**：PWA manifest + 苹果触摸图标。

---

## 5. 无障碍（WCAG AA）
- **对比度**：正文 ≥ 4.5:1，大字 ≥ 3:1；课程调色板已按此校验。
- **键盘 / 焦点**：所有可操作元素 ≥ 44px 触摸目标；焦点环 `2px --brand-500` + `2px` 偏移（`:focus-visible`）。
- **屏幕阅读器**：语义化结构 + ARIA（`role="dialog"` 抽屉、`aria-live` Toast / 横幅、`aria-label` 图标按钮）。
- **文案**：关键操作均有文字标签（不依赖纯图标）；错误信息明确且可操作。
- **减少动效**：`prefers-reduced-motion` 下去位移动画。
- **文本缩放**：布局兼容浏览器 200% 缩放不破版。
